dotfiles
========

This is my dotfiles.

## Requirements

- Homebrew
- zsh >~ 5.0.5
- Vim >~ 7.4
- git >  2.0

NOTE: I usually use this dotfiles on Mac OS X. There may be some problems in other systems.

## How to install

Clone this repository and run `bootstrap.sh`.

```bash
$ cd
$ git clone ${REPOSITORY_PATH} .dotfiles
$ cd .dotfiles

# then just run bootstrap.sh
$ ./bootstrap.sh

# when MacOS
$ cd homebrew
$ brew bundle
```
